# Ward-Projected Hadronic Transport Status

The source harness emits

`Delta_source(P) = Delta_lep(P) + Delta_had(P) + Delta_EW(P)`

in the declared inverse-alpha coordinate and subtraction scheme. It also
reports the dimensionless screening diagnostics derived from the same source
packet.

The calculation uses the internal Stage-5 root, source-chain masses, the
source strong-coupling branch, and the declared perturbative/constituent module
grid. It imports no measured hadronic spectral data or Thomson endpoint.

The following objects are work in progress:

- a source-derived nonperturbative Ward-projected spectral measure;
- a proved electroweak transport contribution;
- a same-scheme physical remainder;
- a certified function or interval over the relevant $P$ domain;
- a complete physical Thomson readout.

The sampled module envelope is a diagnostic. It carries no target, score,
pass/fail rule, promotion weight, or falsification verdict.
