\subsection{Clock-conditioned correlator}

The finite toy model used in the correspondence package has two clock orientations exchanged by

\begin{theorem}[Clock-projection correlator theorem]
Let \(\mathcal H\) be finite-dimensional, let \(H=H^\dagger\), let \(A=A^\dagger\), and set
\[
A(t)=e^{iHt}Ae^{-iHt},\qquad \rho_\infty=I/d.
\]
Then
\[
C_\infty(t):=\Tr(\rho_\infty A(t)A)
\]
is real, so
\[
\im C_\infty(t)=0.
\]
Equivalently,
\[
\Tr(\rho_\infty[A(t),A])=0.
\]
For a noncentral observer-clock record projector \(P_E\),
\[
\rho_E=\frac{P_E\rho_\infty P_E}{\Tr(P_E\rho_\infty)},
\]
the commutator expectation is generically nonzero. In the two-state clock example
\[
H=\frac{\omega}{2}\sigma_z,\qquad A=\sigma_x,
\]
one has
\[
C_\infty(t)=\frac12\Tr(\sigma_x(t)\sigma_x)=\cos(\omega t).
\]
On the forward clock record state \(\rho_+=|0\rangle\langle0|\),
\[
C_+(t)=\langle0|\sigma_x(t)\sigma_x|0\rangle=e^{i\omega t}.
\]
The imaginary semiclassical phase is restored by clock-record conditioning:
\[
\im C_\infty(t)=0,\qquad \im C_+(t)=\sin(\omega t).
\]
\end{theorem}

\begin{proof}
Since \(A(t)\) and \(A\) are Hermitian,
\[
C_\infty(t)^*=\frac1d\Tr(A A(t)).
\]
By cyclicity of the trace,
\[
\Tr(A A(t))=\Tr(A(t)A),
\]
so \(C_\infty(t)^*=C_\infty(t)\), hence \(C_\infty(t)\in\mathbb R\). The commutator statement
follows from \(\Tr([A(t),A])=0\).

For a projected state \(\rho_E\), the projector sits between the state and the clock-transition
operator. The cyclic cancellation above applies only in the commuting case. Generic observer-clock
records are noncentral, so the antisymmetric part can survive. For the two-state witness,
\[

=\cos(\omega t)\sigma_x-\sin(\omega t)\sigma_y.
\]
Then
\[
\sigma_x(t)\sigma_x=\cos(\omega t)I+i\sin(\omega t)\sigma_z.
\]
The maximally mixed trace gives \(\cos(\omega t)\). The \(|0\rangle\) matrix element gives
\(\cos(\omega t)+i\sin(\omega t)=e^{i\omega t}\).
\end{proof}

\subsection{Time-reversal holonomy}

\(\epsilon_i\) with \(g_{ij}=\epsilon_i^{-1}\epsilon_j\) on every overlap.
\end{theorem}

\begin{proof}

\(\Z_2\), the value \(+1\) preserves orientation and the value \(-1\) reverses it.

\end{proof}
